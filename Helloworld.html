用Github保存自己的网站

完整的教程请参见视频教程 Github 的官方网站 ：Github

申请账号

Github官网上可以申请账号,申请后可阅读一下介绍，也可以不读...，我下载了它说要下的Github，Git shell，据说是图形化界面，个人认为这里敲代码比较好用

下载Git bash

Window版网站git-for-windows.github.io 也可以Google一下

创建站点

用账号登陆Github,右上角有一个加号，点击选New repository, 然后填Name，比如Webtest，Description任意，选public，打钩创建README，其他默认就行，create repository

处理站点基本信息

创建好了站点，点击一下README.md的链接，之后右边会有一排按钮，Raw Blame History，点击那一行的笔，进入编辑状态，支持任何html（相当于在标签下编写就行了，初学的话不编辑也没问题）编辑好以后返回你的站点（上述Webtest）页面右下角有个HTTPS clone URL 的，把网址复制下来

上传网页（假设你写好了一个Helloworld.html要上传）

先建一个文件夹，如D:/WWW （待会你的站点webtest就放WWW里了） 打开Git bash(这是一个支持Linux命令的东西)
导航到WWW，输入cd /d/WWW
下载（克隆）站点git clone +上面要复制的网址
这时WWW下多出了Webtest文件夹
cd webtest 进入站点文件夹
cat README.md 可查看README.md文件
现在，把你的网页复制到webtest里！！！
敲入git checkout --orphan gh-pages
敲入git add -A
敲入git commit -a -m "commit1"
如果是第一次操作，请输入 git config --global user.email "你注册的邮箱"
git config --global user.name "你的用户名"

最后 git push origin gh-pages 系统要求你输入用户名以及密码 大工告成

访问你的网页

http://你的用户名.github.io/webtest/Helloworld.html （io后面就相当于你的站点了）

注意，第二次修改时不是每一步都要重做，那些创建东西的步骤就不用重做了。其实只要替换掉html后在站点文件夹下重复，敲入 git commit -a -m "commit3"（"commit3"每次都不一样，第一次写commit1，第二次就写commit2，或其他字符。）然后 git add -A 以及git push origin gh-pages三步即可,若更新成功，会打印好几行字符，有的时候状况比较多，可以试一下多次执行这几条命令。

由于浏览器的缓存功能，更新成功了之后清一下浏览器缓存再访问你的网址才看得出效果，或用另一个浏览器，或等一会再访问

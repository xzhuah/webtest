Here, I would like to provide some of my programming works. The brief introduction and the links for downloading the programs are listed below. The source code of them are also provided. These programs are mainly designed to help students to work efficiently and take some relax.

Dog of war
This is a game I developed from the assignment of COMP 1021. It is written in Python. The original version of this game has some background sound which I didn't include here because they are too large.
War 2
This game is developed with Python by myself. You need to produce some soldiers and control them to defeat the computer. The computer will use different tactics based on the situation. Details of the controlling methods can be found in the comment of the source code. The original version of this game has some background sound which I didn't include here because they are too large.
MySimpleNote8.0
This is a desktop application wrote in Java. This application have a lot of functions which are too many to mention all of them here. The major task of it is to help people make some e-note. It can save color information which made it more powerful than Notepad (To dye the text, you need to first select the text and then select a kind of color). This application can help you do html program, made a file tree for you, open the files which are dragged into its window....The source code of it can be downloaded here.
矩阵计算
This is a desktop application wrote in Java. The major service it provides is to help students calculate the echelon matrix of an arbitrary matrix or the inverse matrix of a square matrix. A big advantage of this program is that it can show you the detailed process. The source code of this program can be found here.
